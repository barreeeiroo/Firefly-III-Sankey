# Firefly III Sankey

Small script which allows to retrieve Firefly III data through its API and output in a Sankey Diagram format.

### Acknowledgements

Highly inspired by the original but now deprecated [Firefly-III/Sankey](https://github.com/firefly-iii/sankey) written
in PHP by JC5.

This tool was built with the assistance of Generate AI. All final decisions, content, and project direction were made
by the repository owner.
